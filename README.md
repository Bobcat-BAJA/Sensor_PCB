To run the sensor board, first, upload the sensor_pcb_teensy_code.ino to the teensy board through the arduino IDE. Then, after testing is completed, puload the data_reader.ino to the teensy through the IDE. This will output the readings to the serial monitor in the Arduino IDE. Copy these readings to a text file and save as a .csv file. Then, upload the .csv as a table to Matlab. Finally, run the sensor_graphing_baja.m in Matlab to graph the results ofr the test. To get more readings, repeat the process fully, including reuploading sensor_pcb_teensy_code to the teensy to begin.
